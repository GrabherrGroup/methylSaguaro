Instructions for running the pipeline

There are 3 steps for doing the analysis with methylSaguaro:

1. merging the data and formatting it according to methylSaguaro
2. Running Saguaro
3. Extracting the results from the output for further analysis and generating PCA plots.

Start by downloading the pipeline file, unzip it and then change the directory to the unzipped folder, i.e. "cd pipeline".
We have used TCGA data so the first step is to merge all the files (located in myfolder) and restructure it to match with Saguaro readable format.

./merge.sh myfolder output [pattern]
myfolder: the folder that contains all the files with beta values
cancerMerged: the name of the file that will contain the merged, formatted data
pattern: choose 0 to select tumor types, 1 to select normal types and 2 to select control samples. If nothing is set all samples are selected.
Now you are ready to run Saguaro following the instructions here.

When the Saguaro computations are complete,  run the following script to annotate the differentiating regions and generate distance matrices.

perl processMethylSaguaroOutput.pl -i inputFile -s saguaroResults -c CpGpositions -d delimiter -o outputFolder
inputFile: the file which was used as the input for Saguaro
saguaroResults: the folder containing Saguaro results
CpGpositions: 450k_dictionary.csv or 850k_dictionary.csv depending on the array
delimiter: the delimiter of CpGpositions file
outputFolder: the folder that will contain the results
